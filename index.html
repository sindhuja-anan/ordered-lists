1. What is HTML?

    • HTML stands for HyperText Markup Language.

      It is the standard language used to create and design webpages.

    • Simple Definition:

    • HTML is the language used to structure content on the web -

      like headings, paragraphs, images, links, and more.

    • Key Points:

      • HyperText: Text with links to other pages (like a hyperlink).

      Markup: Tags that "mark up" or structure the content.

      HyperText: A set of rules and syntax used to write code.

  2. What is difference between HTML and XHTML?

    • HTML (HyperText Markup Language)

    • Not very strict with syntax

    • Browsers are forgiving of errors (like missing tags)

    • Tags can be in uppercase or lowercase

    • Self-closing tags like or don't need a / at the end

    • Commonly used in most web pages

    • XHTML (eXtensible HyperText Markup Language)

    • Based on XML, so it is very strict

    • Browsers are not forgiving — even small mistakes can
      break the page

    • All tags must be lowercase

    • Elements must be properly nested

    • Needs a DOCTYPE declaration and must be well-formed XML

  3. What is tag?

    • In HTML (Hyper Text Markup Language), a tag ia a piece of code

      used to define elements on a web

    • Basic Defination

    • A tag is enclosed in angular brackets< > and usually comes in pair:

    • Open tag: < tagname>

      Closing tag: < / tagname>

  4. What is element?

    • An HTML element is a basic building block of a webpage. It represents

      a piece of content — like a heading, a paragraph, an image, a link, etc.

  5. What is the difference between HTML4 and HTML5?

    • Differences Between HTML HTML5:

    • HTML4 uses a long doctype.

    • HTML5 uses a short one.

    • HTML4 does not support audio and video.

    • HTML5 supports both.

    • HTML4 has fewer tags.

    • HTML5 introduces new semantic tags.

    • HTML4 forms have limited input types

    • HTML5 adds modern input types.

    • HTML4 requires plugins for graphics

    • HTML5 supports canvas and SVG.

    • HTML4 is not mobile-friendly

    • HTML5 is mobile and responsive design friendly.

    • HTML4 uses only cookies for storage

    • HTML5 adds localStorage and sessionStorage.

    • HTML4 has inconsistent error handling

    • HTML5 handles errors better.

    • HTML4 is used in older websites

    • HTML5 is used in modern websites.

    • HTML5 improves performance, accessibility, and browser compatibility.

  6. What are lists?

    • Lists are used to display items one after another

    • Lists in HTML:

    • There are three types of lists in HTML:

      1. Ordered List
      2. Unordered List
      3. Description List

    • Ordered List:Ordered List displays items with numbers (1, 2, 3...).

    • Unordered List:Unordered List displays items with bullets (●).

    • Description List:Description List is used to show terms and their definitions.

    • Lists can be nested (one list inside another list)

    • Lists help organize content clearly and neatly on a webpage.

  7. What is difference between XML and HTML ?

    • HTML

    • HTML is used to display data

    • HTML has predefined tags

    • HTML focuses on how data looks (presentation).

    • HTML is not strict with rules.

    • In HTML, closing tags are sometimes optional.

    • HTML is designed for webpages.

    • HTML ignores small errors in code.

    • HTML is mostly for humans to read.

    • HTML tags define layout and structure of content.

    • HTML is used mainly for web development.


    • XML

    • XML is used to store and transport data.

    • XML allows you to create your own tags.

    • XML focuses on the meaning of data (structure).

    • XML is very strict with syntax rules.

    • In XML, all tags must be properly closed.

    • XML is designed for data sharing between systems.

    • XML does not allow any errors.

    • XML is for both humans and machines to read.

    • XML tags define custom data structures.

    • XML is often used with APIs, web services, and configurations.
